# Task Sprint

A full-stack task marketplace platform (similar to TaskRabbit) where users can post tasks and apply to complete them as service providers.

## Live Demo

[[Live Demo](https://task-sprint-production-e5a3.up.railway.app/)](https://task-sprint-production-e5a3.up.railway.app/)

> Hosted on Render's free tier — the app may take 30-60 seconds to load on first visit if it has been inactive.

## Tech Stack

- **Backend:** Laravel 11, PHP 8.4
- **Frontend:** React, TypeScript, Inertia.js, Tailwind CSS
- **Database:** SQLite / PostgreSQL
- **Admin Panel:** Filament
- **Other:** Spatie Laravel Permission, multilingual support (EN/RU/UK)

## Features

- Post tasks and apply as a service provider
- Category and sub-category browsing
- User roles and permissions (customer, employee, admin)
- Admin panel for managing categories, users, tasks, and reviews
- Multilingual interface (English, Russian, Ukrainian)
